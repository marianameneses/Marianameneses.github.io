-Colocaria as fotmatações do arquivo html.
EX:


"< html lang="pt-br">
< head>
< meta charset="UTF-8">
< meta http-equiv="X-UA-Compatible" content="IE=edge">
< meta name="viewport" content="width=device-width, initial-scale=1.0">
< title>Atividade 5
< /head>"


Dessa forma formata a página definindo os dados que não podem ser definidos por outros elementos. Por exemplo a meta charset="UTF-8"que é uma tag meta utilizada para comunicar aos navegadores qual é o formato de codificação de caracteres utilizado naquele documento.

-Adicionaria um arquivo css para estilização.

Pois assim facilitaria a estilização, podendo estilizar mais coisas ao mesmo tempo. Facilitando também a leitura do código.

-Separaria a tag "img" da tag "h1".

Assim organizaria melhor o texto e evitaria problemas com a estilização.

-Colocaria o atributo "Alt" na tag "img".

Este atributo define um texto alternativo que descreve a imagem. Os Usuários irão ver o texto se a URL da imagem estiver errado, a imagem não está em um dos formatos suportados ou até a imagem ser baixada .

-Mudaria a ordem da tag "i".

A tag "i" precisa ser fechada antes da tag "b". Se uma tag é aberta dentro de outra ela precisa ser fechada antes da primeira.
